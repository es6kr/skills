# Migrate (renamed → to-ghq)

This topic was **renamed to [to-ghq](./to-ghq.md)** for clarity — it pairs with the new [to-bare](./to-bare.md) topic (the inverse direction).

- `migrate` / `to-ghq` — bare+worktree → regular `.git` at the ghq path
- `to-bare` — regular repo → bare + worktree

`/git-repo migrate` still routes here for backward compatibility; follow [to-ghq.md](./to-ghq.md) for the procedure and script (`scripts/repo-to-ghq.sh`).
