# git-repo

Git repository and worktree management with SourceGit integration — clone via ghq, inventory/reuse/create worktrees, convert repos to bare, and handle multi-account SSH and credentials.

## Installation

```bash
npx skills install git-repo
```

Browse on ClawHub: <https://clawhub.ai/skills/git-repo>

### Peer skills

`git-repo` depends on `commit-tidy`. Install it too:

```bash
npx skills install commit-tidy
```

## Usage

Invoke with `/git-repo <topic>` (for example `/git-repo worktree`). See [`SKILL.md`](./SKILL.md) for the full topic list.
